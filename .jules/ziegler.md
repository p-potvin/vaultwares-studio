## 🛡️2023-10-24 - Argument Injection in External Tool Executions
**Vulnerability:** The `VideoAgent` and `ReconstructionAgent` execute external command-line tools like `ffmpeg`, `ns-process-data`, and `ns-train` via `subprocess.run` where user-provided arguments such as `source`, `output`, and `output_dir` were unvalidated and interpolated directly into command parameters, potentially allowing for argument injection.
**Learning:** Argument injection can happen even when utilizing an array for arguments in `subprocess.run` if unvalidated user inputs that begin with `-` are passed. External binaries interpret these as options or flags rather than valid input, opening the possibility for command manipulation.
**Prevention:** Always parse and explicitly reject user-provided argument values that start with a hyphen (`-`) where simple paths or positional strings are expected. Validation checks should optionally allow them if they represent legitimately parsed negative numbers that the binary expects for a specific parameter.
